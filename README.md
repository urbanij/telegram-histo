# telegram-histo

Extract minimal statistics<sup>1</sup> from your Telegram chats.

Inspired by [this](http://pepijndevos.nl/2018/12/25/visualizing-your-whatsapp-conversations.html) blog post, this script tries to achieve a similar objective targeting my Telegram group chat.
